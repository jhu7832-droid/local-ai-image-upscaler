常见问题

Questions

为什么不做成在线上传工具？ Why not make it an online upload tool?

核心目标是隐私和零云端成本。在线上传会改变这个项目的定位。

The goal is privacy and no cloud cost. Uploading would change the promise.

没有 GPU 可以用吗？ Can I use it without a GPU?

可以。快速模式不需要 GPU，AI 高清模式失败时也会自动回退。

Yes. Fast mode does not need a GPU, and AI mode falls back when needed.

输出图片在哪里？ Where are output images saved?

默认在项目的 output 文件夹，文件名会自动添加 upscale 后缀。

By default, outputs are saved in the output folder with an upscale suffix.
