This website is only a project landing page. It does not receive, store, or process images. All image processing runs locally on your own computer.

Supports single images and batch processing from an input folder.

常见问题

FAQ

为什么不做成在线上传工具？ Why not make it an online upload tool?

核心目标是保护隐私和避免云端成本。在线上传会让图片进入服务器，不再是本地工具。

The goal is privacy and no cloud cost. Online uploads would send images to a server, which changes the product promise.

没有 GPU 可以用吗？ Can I use it without a GPU?

可以。快速模式不需要 GPU；AI 高清模式不可用时也会自动切换到快速模式。

Yes. Fast mode does not need a GPU, and AI mode falls back to fast mode when unavailable.

4x 为什么会慢？ Why is 4x slower?

4x 会让像素数量变成原来的 16 倍。大图建议先试 2x。

4x creates 16 times more pixels. For large images, try 2x first.

输出图片在哪里？ Where are output images saved?

默认保存在项目的 output 文件夹，文件名会自动添加 upscale 后缀。

By default, outputs are saved in the output folder with an upscale suffix.

iPhone 可以直接处理图片吗？ Can iPhone process images directly?

目前不建议直接在 iOS 上运行。这个项目是 Python 本地桌面工具，最佳使用方式是在 Mac 或 Windows 电脑上运行，再从手机传入图片。

Not directly for now. This is a local Python desktop tool, best used on Mac or Windows. Transfer iPhone images to a computer first.

Windows 能用吗？ Does it work on Windows?

可以。依赖库 Pillow、OpenCV、NumPy、FastAPI 和 Uvicorn 都支持 Windows。安装 Python 时记得勾选 PATH。

Yes. Pillow, OpenCV, NumPy, FastAPI, and Uvicorn support Windows. Remember to add Python to PATH during installation.
